# HPC Server Usage Application System Prototype

This project is a prototype for the **HPC server usage application system**. It focuses on the development of a robust system for managing HPC (High-Performance Computing) resources, including tasks like application management, security, and testing.

## Features

- **Development of the database** for the HPC server usage application system prototype using **PostgreSQL**.
- **Backend development** using the **Spring framework**, which handles the application's core logic.
- **Frontend development** using HTML + Thymeleaf
- **Application security** implemented with **Spring Security**, including:
  - Authentication
  - Authorization
  - Password encryption
- **Testing and code quality improvement**:
  - Testing the system with the **JUnit** library.
  - Code quality enhancements with **SonarQube**.

## Technologies Used

- **Backend**: Spring Framework
- **Database**: PostgreSQL
- **Frontend**: Thymeleaf
- **Security**: Spring Security
- **Testing**: JUnit, SonarQube
